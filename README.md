# BalmJS template - `svelte`

> A Svelte project

## Requirements

```sh
npm install -g balm-core balm-cli
# OR
yarn global add balm-core balm-cli
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 4.0.0
# balm-core: 4.0.0
```

## Installation

```sh
balm init svelte my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```
